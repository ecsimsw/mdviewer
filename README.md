# mdviewer

Markdown to styled HTML viewer. Opens in browser with zoom controls and PDF export.

## Install

```bash
npm install -g @ecsimsw/mdviewer
```

## Usage

```bash
npx @ecsimsw/mdviewer README.md
npx @ecsimsw/mdviewer ./docs
npx @ecsimsw/mdviewer README.md --out ./export
```

- Opens styled HTML in your default browser
- `−` / `+` buttons to adjust zoom (50%–200%)
- `PDF` button to save as PDF via browser print
